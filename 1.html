HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications. Web vrowsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a webpage semantically and originally included cues for the appearance of the document.
